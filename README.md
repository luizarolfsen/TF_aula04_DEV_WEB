# TF_aula04_DEV_WEB
Exercício TF aula 4 - Node.js: Funções e Tipos de Referência


Você deve criar um arquivo chamado tf-04.js que, ao ser executado com node tf-04.js, mostre no console uma pequena sequência de exemplos sobre funções.

Regras:

tudo deve estar em um único arquivo

o nome do arquivo deve ser obrigatoriamente tf-04.js

o arquivo deve rodar sem erro com node tf-04.js

os exemplos devem ser curtos, claros e executáveis

cada parte deve imprimir algo no console para demonstrar o conceito

criar um repositório público no GitHub

subir o arquivo tf-04.js no repositório

enviar o link do repositório no formulário da disciplina
